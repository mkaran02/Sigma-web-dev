Core Web Vitals

CLS(Commulative Layout Shift)

Unexpected layout shift disrupt user experience. GOOD CLS SCORE=0.1 or less

LCP(Largest Contextal Paint

It reports render time of largest image or text block visible in viewport,realtive to when user first navigsted to page GOOD LCP SCORE=2.5 SEC

FID(First Input Delay)

FID measures the time from when a user first interacts with a page (that is, when they click a link, tap on a button, or use a custom, JavaScript-powered control) to the time when the browser is actually able to begin processing event handlers in response to that interaction good FID score=100 millisec
